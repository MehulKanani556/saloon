# Glow & Elegance — Frontend

React + Vite frontend for the Glow & Elegance salon management system.

See the root [README.md](../README.md) for full project documentation, setup instructions, and API reference.

## Quick Start

```bash
npm install
npm run dev
```

Runs on `http://localhost:5174`

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
